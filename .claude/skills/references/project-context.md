# Project Context Template

このファイルは `project-context.md` の雛形です。各リポジトリの `.claude/skills/references/project-context.md` に配置し、プロジェクト固有の情報を記入してください。

共通スキル（plan, implement, review）はこのファイルを参照して、プロジェクト固有の調査・実装・レビューを行います。
